#########
# Church Aspects with Infuse Province Effects
# If you want to add a church aspect that lets you infuse a province with chi, there are multiple places to edit
# Search for TRANS_CHURCH_ASPECT_INFUSE
# 1. Add an effect on granting the aspect
# 	example: transmutative_path_track_aspect = { aspect = transmutative_path_through_inaction_action }
# 2. Add a trigger for transmutative_path_valid_slot_trigger
# 	file: anb_scripted_triggers_transmutative_path.txt
# 3. Add a section in transmutative_path_clear_aspect_slot
# 	file: transmutative_path_scripted_effects.txt
# 4. Add the actual infuse effect in transmutative_path_infuse_province_effect
# 	file: transmutative_path_scripted_effects.txt
#########
